# TestEmployeeApi

## Project Structure

A brief overview of the main folders in this project:

- **Controllers**  
  Contains API controllers and defines endpoints for the application.

- **Data**  
  Holds the application database context (`DbContext`).

- **Dtos**  
  Contains Data Transfer Objects (DTOs) used for transferring data between layers.

- **Helpers**  
  Includes helper classes, such as query objects or utility functions.

- **Interfaces**  
  Contains interfaces for repositories, services, and other abstractions.

- **Mappers**  
  Provides mapper classes that convert models to DTOs and vice versa.

- **Repository**

- **Services**
